# Synapsepunctaanalysis
Measure synapse puncta density, size, intensity and colocalization


Identifying the underlying mechanisms of synapse loss in AD is a critical step toward developing effective therapeutic strategies. 

This Fiji macro can be used to select and crop regions of interest for neuronal process segments and measure their length. On the cropped images, we use a macro with the comDET plugin to quantify synapse number, size, and intensity. Synapse density is then calculated by dividing the number of synaptic puncta per 50 μm of axonal length. We used this protocol to quantify synapse loss with neuronal aging, other risk genes, and in NRF2 KO mice (10.1016/j.celrep.2025.116005; 10.1016/j.redox.2025.103853)
